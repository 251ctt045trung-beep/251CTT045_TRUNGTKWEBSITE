
ĐỀ TÀI: WEBSITE QUÁN CÀ PHÊ

GIỚI THIỆU DỰ ÁN
Dự án xây dựng website giới thiệu quán cà phê với các chức năng cơ bản như:
Trang chủ giới thiệu quán.
Danh sách sản phẩm/đồ uống.
Thông tin khuyến mãi.
Thông tin liên hệ và địa chỉ quán.
Giao diện thân thiện, dễ sử dụng trên nhiều thiết bị.
BÁO CÁO QUÁ TRÌNH THỰC HIỆN

2.1. Công cụ AI đã sử dụng (nếu có)

ChatGPT: hỗ trợ tham khảo ý tưởng giao diện, bố cục website và nội dung mô tả.
GitHub Copilot (nếu có): hỗ trợ gợi ý mã nguồn HTML/CSS/JavaScript.

2.2. Các prompt chính đã sử dụng

"Thiết kế giao diện website quán cà phê hiện đại bằng HTML và CSS."
"Gợi ý bố cục trang chủ cho website quán cà phê."
"Viết nội dung giới thiệu ngắn cho quán cà phê."
"Tạo menu đồ uống mẫu cho website quán cà phê."

2.3. Những phần tự thực hiện

Phân tích yêu cầu và lên ý tưởng website.
Thiết kế giao diện bằng HTML/CSS.
Xây dựng cấu trúc các trang web.
Tùy chỉnh màu sắc, hình ảnh và bố cục.
Kiểm tra và sửa lỗi giao diện.
Đưa dự án lên GitHub Pages.

2.4. Những phần tham khảo AI

Tham khảo ý tưởng bố cục giao diện.
Tham khảo nội dung giới thiệu quán.
Tham khảo một số đoạn mã CSS và JavaScript mẫu.
Tham khảo cách tối ưu giao diện responsive.

2.5. Khó khăn gặp phải

Sắp xếp bố cục giao diện sao cho đẹp và dễ sử dụng.
Điều chỉnh giao diện tương thích trên điện thoại và máy tính.
Tìm kiếm hình ảnh phù hợp với chủ đề quán cà phê.
Khắc phục một số lỗi hiển thị CSS.
KẾT QUẢ ĐẠT ĐƯỢC
Hoàn thành website giới thiệu quán cà phê.
Giao diện trực quan, dễ sử dụng.
Website có thể truy cập trực tuyến thông qua GitHub Pages.
